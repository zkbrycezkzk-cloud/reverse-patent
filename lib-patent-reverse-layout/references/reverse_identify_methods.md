# 可反向性能测试 / 产品鉴定方法库 (Reverse Identification Methods)

本文档定义反向专利布局技能 Step 4 阶段使用的产品鉴定方法库。目标：撰写说明书"产品鉴定方法"章节，使**拿到一只未知来源的锂离子电池后，能够通过拆解、取样、净化和定量表征，逐步得到权利要求所限定的全部参数**。

---

## 一、产品鉴定方法的核心价值

### 1.1 与"详细性能测试"的区别

| 维度 | 详细性能测试（Step 3） | 产品鉴定方法（Step 4） |
|---|---|---|
| 测试对象 | 自家做的实施例/对比例 | 未知来源的他人成品 |
| 起点 | 已知材料配方+工艺 | 黑盒：仅有成品电池 |
| 目的 | 验证本发明效果 | 反向举证侵权 |
| 必须解决 | 测什么+怎么测 | **怎么拆 + 怎么取样 + 怎么测** |

### 1.2 反向布局的关键支撑

权利要求若**无法从成品上反查**，则在侵权诉讼中举证困难，专利的实际保护力削弱。因此 Step 4 是反向布局**不可省略的关键环节**——它确保每条权要都"测得出来"。

---

## 二、四层鉴定框架

产品鉴定方法应按以下四层结构撰写：

```
Layer 1: 样品获取与三级分类
Layer 2: 整只电池→正极/负极样品的拆解流程
Layer 3: 按权要逐项的测试方法（三维分类组织）
Layer 4: 核心特征组合判定 → 形成侵权认定结论
```

---

## 三、Layer 1：样品三级分类法

从拆解后的极片应分为三类样品，用于不同测试目的：

| 样品类型 | 制备方式 | 用途 | 优先级 |
|---|---|---|---|
| **A类：原始极片样** | 从洗净后的极片剪取小片 (3×5 mm)，**不超声、不刮粉** | FIB-TEM/SEM 截面观察、界面相、显微拉曼、Raman 映射 | 最高（结构识别核心证据） |
| **B类：洗净粉体样** | 刮取涂层 → NMP 溶解 PVDF → 乙醇洗涤 → 真空干燥 | XRD、ICP-OES、SEM 粒径统计、激光粒度、BET 比表面积 | 高（组成与分布证据） |
| **C类：纯化活性物样** | 在 B 类基础上 → 低速沉降去除游离导电炭 → 真空干燥 | 粉末压实密度、重新制备半电池验证 | 辅助（性能验证） |

**原则**：若电池容量小、正极材料量不足，**优先完成 A 类和 B 类测试**（结构识别核心证据），C 类为辅助验证。

**样品量建议**：
- A 类：至少 5 片小样（不同位置取样）
- B 类：≥ 200 mg 粉体（用于多项测试）
- C 类：≥ 50 mg 纯化活性物

## 四、Layer 2：电池拆解取样标准流程

```
1. 预放电
   ├─ 0.1C ~ 0.2C 放电至截止电压以下
   └─ 未知容量时：放至 2.0 ~ 2.5 V
   
2. 静置 ≥ 2 h（等待残余反应平衡）
   
3. 惰性气氛拆解
   ├─ 设备：Ar 手套箱
   └─ 环境：H2O < 1 ppm，O2 < 1 ppm
   
4. 物理拆解
   ├─ 18650/21700：滚边剪开端盖 → 推出卷芯
   ├─ 软包：剪开铝塑膜 → 分离卷芯/叠片
   └─ 方壳：撬开盖板 → 取出卷芯
   
5. 卷芯展开
   ├─ 拆胶带（先记录胶带位置/尺寸/颜色/材质）
   ├─ 沿卷绕方向逐圈展开
   └─ 记录极片长度、隔膜位置、入位/收尾顺序
   
6. 极片分离
   ├─ 正极片（铝箔集流体）→ DMC 漂洗 3 次（每次 30s ~ 1min）
   ├─ 负极片（铜箔集流体）→ DMC 漂洗 3 次
   └─ 隔膜 → 单独保存（不漂洗）
   
7. 真空干燥
   ├─ 温度：60 ℃
   └─ 时间：12 h
   
8. 样品分类制备（见 Layer 1）
```

## 五、Layer 3：按权要逐项的测试方法

按"材料维度 / 极片维度 / 性能维度"三维分类组织，每项对应一条权要。

### 5.1 极片维度的鉴定

#### 5.1.1 胶带尺寸 (长 × 宽 × 厚)

**适用权要**：包含胶带/绝缘件尺寸限定的权要。

**操作步骤**：
```
1. 拆解过程中拍照记录每条胶带的原始位置（卷芯展开前）
2. 小心剥离胶带（避免拉伸变形）
3. 平整置于无水乙醇中浸泡 30 s（去除粘附残胶）
4. 取出晾干
5. 测量：
   ├─ 长度：游标卡尺，精度 0.02 mm
   ├─ 宽度：游标卡尺，精度 0.02 mm
   └─ 厚度：螺旋测微器或测厚仪（无变形），精度 0.001 mm
6. 同一卷电芯取 3 片同类胶带，取平均值
```

**判定**：取样平均值是否落入权要限定的区间。

---

#### 5.1.2 折返长度 / 展开宽度

**适用权要**：包含 W1/W2、L1/L2 等比值的权要。

**操作步骤**（以展开宽度 W 为例）：
```
1. 在卷芯未展开时，用游标卡尺测量胶带可见展开宽度
2. 展开卷芯后，沿胶带粘贴方向测量总展开宽度（包括跨越极片端边的两面）
3. 计算 W = 总展开宽度
4. 对正极收尾胶带 W1 与负极入位胶带 W2 分别测量
5. 计算 W1/W2 比值
```

**判定**：W1/W2 比值是否落入权要限定的区间（如 0.50 ~ 1.10）。

---

#### 5.1.3 极片几何特征（凹口 / 折弯 / 涂层结构）

**适用权要**：包含极片凹口数量/深度/节距，或涂层单面/双面/跨越关系的权要。

**操作步骤**：
```
凹口测量：
1. 展开极片，平整放置
2. 沿极片纵向方向用游标卡尺测量：
   ├─ 凹口总数 N
   ├─ 凹口深度 D
   ├─ 相邻凹口节距 P
   └─ 凹口圆角半径 R（轮廓投影仪测量）
3. 拍照标记位置，与端部绝缘件距离 d

涂层结构观察：
1. 切取 5 mm × 5 mm 小片
2. 镶嵌树脂 → 截面抛光
3. SEM 观察截面：单面/双面、涂层厚度、是否跨越端边
4. EDS 元素映射验证集流体材质 (Al / Cu)
```

---

### 5.2 材料维度的鉴定

#### 5.2.1 基材种类 (PI / PET / PP / 陶瓷涂层等)

**适用权要**：包含基材马库什限定的权要。

**测试方法**：

| 方法 | 设备 | 关键特征 |
|---|---|---|
| **FTIR-ATR** | Bruker Alpha + ATR | PI: 1720 cm⁻¹ (酰亚胺 C=O 对称伸缩) + 1780 cm⁻¹ (反对称) + 1380 cm⁻¹ (C-N)；PET: 1715 cm⁻¹ (酯 C=O) + 1240 cm⁻¹ (酯 C-O-C)；PP: 2950/2870 cm⁻¹ (CH3/CH2)，1450/1375 cm⁻¹ |
| **DSC** | TA Q200 | PI: 无明显熔点；PET: Tm ≈ 245-260 ℃；PP: Tm ≈ 165 ℃ |
| **TGA** | TA Q500 | PI: T5% ≈ 550 ℃；PET: T5% ≈ 380 ℃；PP: T5% ≈ 300 ℃ |
| **燃烧** | 简易点火 | PI: 难燃，离火即灭，黑色残渣；PET: 自熄，蓝色火焰带黑烟；PP: 易燃，无烟，蜡味 |

**判定**：通过 FTIR 主特征峰确认基材类型；DSC/TGA 作为辅助验证。

---

#### 5.2.2 活性物质 (NCM / LFP / 石墨 / SiOx)

**适用权要**：包含正/负极活性物质种类限定的权要。

**测试方法**：

| 方法 | 设备 | 关键特征 |
|---|---|---|
| **XRD** | Rigaku SmartLab | NCM/NCA: R-3m，(003)/(101) 双峰；LFP: Pnma，特征峰 25.5°/29.6°/35.5°；石墨: (002) 26.5°；SiOx: 28.4° (Si 主峰) |
| **ICP-OES** | iCAP 7400 | 元素含量定量：NCM811 中 Ni:Mn:Co ≈ 8:1:1；LFP 中 Fe/P ≈ 1:1 |
| **SEM-EDS** | Hitachi SU8010 | 元素映射 + 颗粒形貌 |

**判定**：XRD 主峰位与标准卡片 (PDF) 比对 + ICP 定量 + EDS 验证。

---

#### 5.2.3 电解液组成 (LiPF6 / LiFSI / 溶剂 / 添加剂)

**适用权要**：包含电解液组成的权要。

**测试方法**：

| 方法 | 用途 |
|---|---|
| **GC-MS** | 溶剂种类与比例（EC、DMC、EMC、DEC 等） |
| **NMR (¹⁹F)** | 锂盐种类（LiPF6 在 -73 ppm，LiFSI 在 +52 ppm） |
| **IC** | 阴离子浓度（PF6⁻、FSI⁻） |
| **ICP-MS** | Li 浓度，验证盐浓度 |

**注**：电解液取样需在 Ar 气氛下用注射器抽取，避免水解。

---

### 5.3 性能维度的鉴定

#### 5.3.1 模量 / 拉伸性能 (胶带或隔膜)

**操作步骤**：
```
1. 从拆解的胶带/隔膜上裁取标准样条 (50 mm × 10 mm)
2. 万能材料试验机：拉伸速率 50 mm/min
3. 计算拉伸模量 (在 0.05% ~ 0.25% 应变区间)
4. 取 5 条样品的平均值
```

**判定**：模量是否落入权要区间（如 1.5 ~ 6.0 GPa）。

---

#### 5.3.2 热收缩率

**操作步骤**：
```
1. 从原始胶带/隔膜上切取 100 mm × 100 mm 样品
2. 标记 MD/TD 方向，测量初始尺寸
3. 85 ℃ / 30 min 烘箱（按权要选定）
4. 取出冷却 30 min，重新测量尺寸
5. 计算收缩率
```

**判定**：收缩率是否满足权要阈值（如 ≤ 2.0%）。

---

#### 5.3.3 颜色 / 视觉鉴别

**操作步骤**：
```
1. 拆解过程中拍照记录每个组件的颜色（自然光下）
2. 必要时用分光测色仪 (Konica Minolta CM-700d) 测 L*a*b* 坐标
3. 与权要中限定的颜色对比
```

**判定**：颜色—材质组合是否与权要一致（如"黄色 PI 胶带 + 蓝色 PET 胶带"）。

> 视觉鉴别是反向举证中**门槛最低、成本最低**的方法。在反向布局中应充分利用。

---

#### 5.3.4 电化学性能验证 (C 类样品)

**操作步骤**：
```
1. 用 C 类样品 + 标准对电极（Li 金属）+ 标准电解液 + 标准隔膜，重新组装 CR2032 半电池
2. 化成与测试：参考 reverse_perftest_methods.md 第 2 节
3. 测试项：比容量、首效、倍率（可选）
```

**注**：电化学性能由电池整体设计决定，不直接证明侵权。但可作为辅助证据，证明活性材料/电解液与权要中限定的种类一致。

---

## 六、Layer 4：核心特征组合判定

完成所有逐项测试后，按以下逻辑形成侵权判定结论：

```
判定步骤：
1. 列出权要 1（独立权要）的全部技术特征
2. 逐项对照测试结果
3. 若全部命中 → 落入独立权要保护范围，构成侵权初判
4. 进一步对照从属权要，确认侵权深度
5. 视觉鉴别 (颜色) + 卡尺测量 (尺寸) → 初步筛查
6. 仪器测试 (FTIR/XRD/模量) → 二次确认
7. 性能测试 (热收缩率/电化学) → 最终验证
```

**判定结论标准句式**：

> 若所测样品同时满足以下条件，则可初步判定其落入本专利权利要求 1 的保护范围：
> ① [核心特征 1]
> ② [核心特征 2]
> ③ ...
> 
> 进一步若同时满足以下条件，则落入权利要求 [N] 的保护范围：
> ① [从属特征 1]
> ② ...
