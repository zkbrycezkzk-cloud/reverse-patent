# 三维设计模型 (Three-Dimension Design Model)

本文档定义反向专利布局技能中权利要求构造的底层模型。每件锂电产品同时从**材料维度**、**极片维度**、**性能参数维度**三个维度被切片，权利要求可以是任一维度的单一限定，也可以是多维度的耦合限定。

---

## 一、三个维度的定义

### 1.1 材料维度 (Material Dimension)

指构成产品的物质组成与化学/物理基础属性，包括：

| 子类 | 锂电常见特征 |
|---|---|
| 基材种类 | PI膜、PET膜、PP膜、PE膜、芳纶、陶瓷涂层、PVDF、CMC、SBR |
| 活性材料 | LCO、NCM、NCA、LFP、LMO、石墨、硬碳、SiOx、Si-C复合物 |
| 电解液组成 | EC、DMC、EMC、LiPF6、LiFSI、VC、FEC 等添加剂 |
| 化学组成 | 元素掺杂量、化学计量比、纯度、痕量元素 |
| 牌号 / 来源 | 商业牌号（如 NCM811、PVDF 5130 等） |

**特征**：通常用马库什式（Markush）撰写，可层层上位（PI → 含酰亚胺基的高分子膜 → 耐高温有机膜）。

---

### 1.2 极片维度 (Geometry / Structure Dimension)

指产品的几何结构、尺寸、位置、形态、连接方式，包括：

| 子类 | 锂电常见特征 |
|---|---|
| 尺寸参数 | 长、宽、厚（极片/胶带/隔膜/集流体）；直径、高度（颗粒、电芯） |
| 比值参数 | 厚度比、宽度比、长宽比、含量比 |
| 位置参数 | 入位 / 收尾；正极侧 / 负极侧；端部 / 中间 / 折返 |
| 几何特征 | 凹口（数量、深度、节距、圆角 R）、孔（孔径、孔密度）、台阶、斜面 |
| 涂层结构 | 单面 / 双面、跨越 / 不跨越、连续 / 间断、单层 / 多层 |
| 装配关系 | 卷绕 / 叠片 / 卷叠混合、极耳焊接位置、隔膜伸出量 |

**特征**：通常用具体数值范围撰写，是反向布局中**最重要的可鉴定参数族**。

---

### 1.3 性能参数维度 (Performance Dimension)

指产品在特定测试条件下表现出的可量化性能，包括：

| 子类 | 锂电常见特征 |
|---|---|
| 电化学性能 | 比容量、首效、倍率（1C/3C/5C）、循环保持率（100/500/1000周）、内阻、EIS |
| 热性能 | 热收缩率（85/120/150℃）、热分解温度（DSC）、热扩散系数 |
| 力学性能 | 拉伸模量（GPa）、拉伸强度（MPa）、断裂伸长率、剥离强度（N/m）、压实密度 |
| 化学/电化学稳定性 | 耐电解液性、耐高电压、电化学窗口（V）、SEI 稳定性 |
| 安全性能 | 针刺、过充、热冲击、挤压（通过/起火/爆炸的判定） |
| 视觉/光谱鉴别 | 颜色（黄/蓝/白/黑）、FTIR 特征峰、Raman 特征峰、XRD 衍射峰位 |

**特征**：通常用「不大于 / 不小于 / 在 A 至 B 之间」等阈值或区间表达；颜色等视觉特征可直接列举。

---

## 二、维度组合的 6 种模式

### 模式 1：单材料维度

> 「所述第一端部绝缘件的基材为聚酰亚胺（PI）膜」

适用：上位概括材料种类、布局材料替代物的保护。

---

### 模式 2：单极片维度

> 「所述第一端部绝缘件的展开宽度 W1 为 7.5 mm ~ 12.5 mm」

适用：保护可直接测量的几何参数。**反向布局中最常用的独立权要构造方式**。

---

### 模式 3：单性能维度

> 「所述第一端部绝缘件在 85℃、30 min 条件下的热收缩率不大于 2.0%」

适用：保护功能性效果，作为材料/极片维度的协同限定。

---

### 模式 4：材料 × 极片耦合

> 「所述第一端部绝缘件的基材为聚酰亚胺（PI）膜，且其展开宽度 W1 为 7.5 mm ~ 12.5 mm」

适用：把材料种类与几何参数绑定，提高创造性、增加规避难度。

---

### 模式 5：材料 × 性能耦合

> 「所述第一端部绝缘件的基材为聚酰亚胺（PI）膜，且其拉伸模量为 1.5 GPa ~ 6.0 GPa」

适用：把材料与功能性能绑定，常用于性能性权要。

---

### 模式 6：极片 × 性能耦合

> 「所述第一端部绝缘件的展开宽度 W1/W2 为 0.50 ~ 1.10，且在 85℃ 下的热收缩率不大于 2.0%」

适用：把几何参数与功能性能绑定，构造"形态+功能"的复合限定。

---

### 模式 7：三维耦合

> 「所述第一端部绝缘件的基材为聚酰亚胺（PI）膜，展开宽度 W1/W2 为 0.60 ~ 0.90，且拉伸模量不小于 1.5 GPa」

适用：精准卡死产品特征组合，几乎不可规避，但保护范围窄。

---

## 三、维度选择策略

### 3.1 独立权要的维度选择

| 布局路线 | 独立权要维度 | 优点 | 缺点 |
|---|---|---|---|
| **攻击型** | 单极片维度（最易测量） | 保护范围最宽 | 易被无效（创造性受质疑） |
| **防御型** | 三维耦合 | 难以规避 + 稳定授权 | 保护范围窄 |
| **平衡型**（推荐默认）| 极片维度 + 一项辅助限定 | 兼顾宽度与稳定性 | — |

### 3.2 从属权要的维度递进

推荐递进路径（由宽到窄）：

```
独权 (极片维度，宽范围)
  ↓
+ 优选范围            （仍单维度）
  ↓
+ 性能维度限定         （引入耦合）
  ↓
+ 材料维度限定         （进一步耦合）
  ↓
+ 视觉鉴别 / 具体值    （最具体，对应实施例）
```

## 四、可鉴定性优先原则

在选择维度参数时，**易测量的优先**。可鉴定性 5 档：

| 等级 | 测量手段 | 示例 |
|---|---|---|
| ★★★★★ | 卡尺 / 目视 | 极片尺寸、胶带宽度、颜色 |
| ★★★★ | 天平 / 螺旋测微器 | 厚度、重量、密度 |
| ★★★ | 常规仪器（XRD、SEM、FTIR） | 晶相、形貌、官能团 |
| ★★ | 拆解后专业测试（EIS、DSC、TGA） | 电化学阻抗、热行为 |
| ★ | 破坏性测试（针刺、爆炸） | 安全性能 |

**反向布局中，独立权要的核心参数应优先选 ★★★★★ 或 ★★★★ 级**。

## 五、锂电常见特征的三维归属速查表

| 实测特征 | 归属维度 |
|---|---|
| 胶带颜色（黄/蓝） | 性能（视觉鉴别） |
| 胶带长 / 宽 / 厚 | 极片 |
| 胶带基材（PI/PET/PP） | 材料 |
| 胶带模量 / 热收缩率 | 性能 |
| 极片凹口数量 / 深度 / 节距 | 极片 |
| 正极活性物质（NCM/LFP） | 材料 |
| 正极涂层厚度 / 压实密度 | 极片 / 性能 |
| 正极比容量 / 倍率性能 | 性能 |
| 隔膜陶瓷涂层厚度 | 极片 |
| 隔膜热收缩率 | 性能 |
| 电解液 LiFSI 浓度 | 材料 |
| 极耳焊接位置 / 数量 | 极片 |
| 卷芯外径 / 圈数 | 极片 |
| 电芯能量密度 | 性能 |
| 电芯循环保持率 | 性能 |
| 电芯针刺通过率 | 性能 |

> 对边界情况（如"压实密度"既涉及几何也涉及性能），归属以**便于权要撰写的维度**为准，不强求唯一性。
