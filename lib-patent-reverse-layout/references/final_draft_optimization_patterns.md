# 最终申请稿整合优化规则

用于 `Reverse_Compile` 已完成后，或用户要求“根据最新最终申请稿继续优化、固化、加入 skill”时。目标是把已确认的权利要求、实施例、测试方法和反向鉴定方法整合为更接近可提交申请文本的说明书表达，同时保持反向可测、数据透明和已确认保护范围不被改窄。

## 一、适用触发

在以下场景优先使用本规则：

1. 用户要求增加实施例1详细实施方式，包括材料选择、配方选择、正负极极片设计和装配化成步骤。
2. 用户要求将实施例和对比例具体化为“制备过程中如何调节参数”。
3. 用户要求性能解释按实施例与对比例逐变量比较，并补充专业机理。
4. 用户要求产品反向鉴定方法写成“拿到一只未知电池后如何一步步定量获得权利要求参数”。
5. 用户要求把已经修改、更新后的最终稿经验固化为 skill 模板。

本规则不替代 Step 1 至 Step 4 的确认流程；若仍处于分段工作流中，应先完成对应 Step 的确认。

## 二、实施例1详细实施方式模板

实施例1应作为实测产品或优选产品样品，写成可复现的制备路径。至少包括以下内容：

1. 正极浆料制备：写明正极活性材料类型、关键元素或颗粒级配、导电剂、粘结剂、溶剂及固体组分质量百分比。
2. 正极极片制备：写明铝集流体厚度、涂布方式、烘干、辊压、分切，以及正极单面面密度、压实密度、极片厚度、扣电克容量和核心计算参数。
3. 负极浆料制备：写明石墨、硅基材料、导电剂、增稠剂、粘结剂、溶剂及固体组分质量百分比。
4. 负极极片制备：写明铜集流体厚度、双面涂布、两面面密度或平均面密度、压实密度、极片厚度、扣电克容量和容量配平口径。
5. 隔膜和卷芯制备：写明隔膜基膜、涂层、总厚度、卷绕顺序、正负极对齐和伸出关系。
6. 装配和化成：写明入壳、集流构件连接、滚槽、注液、封口、化成、老化和分容，最后列出实施例1核心参数和整电性能。

示例句式：

```text
步骤2，正极极片制备。选用厚度tAl为[数值]μm的铝集流体，将步骤1所得正极浆料双面涂布于铝集流体表面，经烘干、辊压和分切后，使正极活性物质层的单面面密度Mp为[数值]mg/cm2，正极极片压实密度为[数值]g/cm3，正极极片厚度为[数值]μm。拆解正极活性材料经25±2℃、0.1C扣电测试得到Qp为[数值]mAh/g，由此得到β=Mp×Qp/1000=[数值]mAh/cm2。
```

## 三、实施例和对比例具体化句式

除实施例1外，其他实施例和全部对比例应尽量写成制备过程中的明确调节，不仅写结果参数。优先采用如下统一格式：

```text
实施例N。本例与实施例1的区别在于：通过调整[材料投料/涂布泵速/涂布间隙/集流体厚度/辊压压力/有效涂布长度/结构件质量/焊接窗口]，使[参数名称]由[原值]变为[新值]，其余条件与实施例1相同。
```

```text
对比例N。本对比例与实施例1的区别在于：通过调整[具体制备动作]，使[偏离参数]变为[偏离值]，其余条件与实施例1相同。
```

撰写要求：

1. 每个例优先对应一个主变量；若必须包含多个变量，应说明各变量分别由何种制备动作引起。
2. 面密度变化优先写为调整涂布泵速、涂布模头间隙或浆料固含量，不笼统写“调整设计”。
3. 集流体厚度变化写为选用不同厚度箔材，不写成测试后得到。
4. 有效反应面积变化写明是调整涂布长度、涂布宽度、分切宽度或留白区域；若只需改变面积倍率，应避免同时放大长度和宽度导致面积倍率不一致。
5. 整电质量变化应写为壳体、盖板组件、绝缘件、电解液保液量或外套膜质量变化。
6. 内阻变化应写为集流构件焊接窗口、接触面积、焊点数量、盖板连接部接触电阻等产品可留下痕迹的调节。
7. 不得用“采用更高克容量材料或调整面密度”这类二选一表达替代明确调节路径；用户已要求单一路径时，应固化为单一路径。

## 四、性能解释段模板

性能解释应从实施例和对比例数据出发，采用“对比对象、变量偏离、性能变化、机理解释、范围结论”的连续论证。推荐格式：

```text
对比实施例[编号]、实施例[编号]与对比例[编号]可知，[变量]用于限定[技术含义]。[变量]偏低时，[材料/结构层面的变化]，导致[离子传输、电子收集、界面反应、机械稳定或热行为]变化，表现为[容量、能量、RAC、δ、温升、循环等结果]。[变量]偏高时，[另一方向变化]，导致[性能劣化机理]。因此，将[变量]限定在[范围]，能够兼顾[至少两个技术效果]。
```

撰写要求：

1. 每个核心变量至少写一段，通常不少于100字。
2. 变量包括但不限于材料组成、颗粒级配、箔材厚度比、面容量、容量配平、质量能量密度、内阻比、产气量、温升、孔隙率、导热系数等。
3. 专业术语必须对应变量类型。例如，正极面容量可解释为孔道曲折度、液相传输阻力、颗粒内部扩散路径、正极极化和界面副反应；负极容量配平可解释为负极电位、析锂风险、SEI膜增长和局部电流密度；集流体厚度比可解释为电子收集截面积、非活性质量、焊接热影响和卷绕张力稳定性。
4. 除物质化学式、实验结果参数和必要公式外，性能解释段不使用箭头类符号。
5. 不用空泛句式替代表格数据，不只写“效果更好”；必须说明为什么数据变化与权利要求范围相关。

## 五、硅容量占比的电压区间容量积分法

当负极包括石墨和硅基材料，且用户要求通过电压区间区分硅贡献容量与石墨贡献容量时，优先采用如下方法：

1. 使用负极0.1C扣电脱锂曲线，导出电压V与容量Q数据。
2. 以石墨标准负极和硅碳标准负极建立电压区间参考。
3. 若未建立标准样，dQ/dV曲线仅作为辅助工具确定分界电压Vb，Vb优选取0.28V至0.32V范围内位于石墨特征贡献区和硅基材料宽电位贡献区之间的局部谷点。
4. 将0.005V至Vb区间的容量增量记为石墨贡献容量Qgr，将Vb至0.75V区间的容量增量记为硅贡献容量Qsi。
5. 0.75V至1.5V的尾部容量不大于总脱锂容量5%时并入Qgr；超过5%时，采用石墨标准负极曲线进行基线扣除。
6. 按硅容量占比=Qsi/(Qsi+Qgr)×100%计算。
7. 用SEM/EDS中Si元素分布、负极总克容量Qn和配方灰分趋势校核。

不得把该方法写成“峰面积分离”作为主计算方式；dQ/dV只作为分界电压辅助判断。

## 六、单只未知电池反向鉴定流程

产品鉴定方法应写成从一只未知成品电池出发的闭环流程。推荐顺序：

1. 完整电芯初始记录和性能复测：记录外观、尺寸、开路电压和质量；按统一充放电制度测容量和能量；按固定SOC、静置时间和频率测RAC；计算γ和δ。
2. 拆解取样和样品保存：将同一只电芯放电至指定截止电压，转入惰性气氛；拆解壳体，展开卷芯，标记卷首、卷中、卷尾和极片两面；清洗干燥并记录取样面积。
3. 集流体厚度：采用CP截面或FIB截面SEM测tAl和tCu，必要时用剥离裸箔称重法校核，计算α。
4. 正极面容量：冲取已知面积正极片，称量总质量和剥离后铝箔质量，计算Mp；组装正极扣电得到Qp；计算β。
5. 负极容量配平：分别测两个工作面面密度并取平均得到Mn；组装负极扣电得到Qn；展开卷芯测Ap和An；计算Cp、Cn和η。
6. 材料组成：用ICP或EDS测元素含量，用SEM统计多晶和单晶颗粒级配，用电压区间容量积分法测硅容量占比。
7. 辅助结构：测极片厚度、压实密度、隔膜总厚度、基膜和陶瓷涂层厚度。
8. 权要映射：将每条权利要求核心特征对应到上述至少一个检测路径。

公式应在鉴定方法中重复出现，避免只在权利要求中定义而鉴定章节无法独立执行。

## 七、最终稿二次优化自检

输出前检查：

| 检查项 | 要求 |
|---|---|
| 实施例1 | 已写清材料、配方、极片、卷绕、装配、化成和核心参数来源 |
| 其他例 | 均采用“本例与实施例1的区别在于”或“本对比例与实施例1的区别在于” |
| 调节路径 | 每个参数变化均能追溯到具体制备动作 |
| 性能解释 | 逐变量对比实施例和对比例，每段说明机理和范围结论 |
| 表格列 | 最终申请稿中的实施例、对比例和性能数据表不保留“数据标识”“规避类型”“证明目标”等内部管理列；需要区分预测值时将`[P]`附着于数值 |
| 确定结果 | 实施例1和关键优选例的安全、热箱或通过性结果写成确定结论，不保留二选一或互斥结果 |
| 反向鉴定 | 从单只未知电池出发，包含完整电芯测试、拆解取样、材料扣电、结构测量和公式计算 |
| 硅容量占比 | 以电压区间容量积分为主，dQ/dV仅辅助确定Vb |
| 符号风格 | 性能解释段不使用箭头类符号 |
| 数据透明 | 预言性数据保留[P]标记，实测与推理不混淆 |
| 脱敏避让 | 不保留商业型号、内部项目名、用户要求避开的旧案核心主线 |
