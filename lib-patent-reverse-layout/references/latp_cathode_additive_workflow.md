# LATP正极添加剂产品保护布局专项规则

本文件用于用户提供LATP作为锂离子电池正极添加材料、陶瓷颗粒、正极功能添加剂或类似实测/预测数据时。其目标是把附件中的材料本体参数、正极添加比例、极片参数和电芯性能组合为可反向检测、可支撑创造性的产品权利要求和说明书实施例。

## 1. 核心布局原则

1. 权利要求核心不以浆料固含量、pH、黏度作为必要参数。上述参数只可作为加工稳定性、分散性或制备窗口写入说明书，不进入核心独立权利要求。
2. 独立权利要求优先采用“LATP陶瓷颗粒材料参数+正极中添加量+电芯性能”的耦合式保护。
3. 性能指标用于创造性证据和从属限定，不单独支撑独立权利要求；必须回扣到可反向检测的材料参数或极片参数。
4. 每个核心参数必须能通过成品拆解、材料留样或行业常规测试获得，优先选择SEM/EDS、ICP、XRD/Rietveld、Karl Fischer、BET、振实密度、磁性杂质、EIS或阻抗谱。
5. 缺失材料参数可在用户确认后使用合理预测值，但阶段稿必须保留`[P]`标记；正式Word或提交版应按用户要求统一清除或转化为常规实施例表述。

## 2. LATP参数优先级

权利要求1和核心从权优先从以下参数中选择：

| 参数族 | 推荐参数 | 检测路径 | 写入层级 |
|---|---|---|---|
| 粒径分布 | D10、D50、D90、D90/D50 | 激光粒度、SEM统计 | 独权或核心从权 |
| 晶相 | NASICON相含量、杂相含量 | XRD/Rietveld | 独权或核心从权 |
| 组成 | Li/Al/Ti/P元素摩尔比、Li1+xAlxTi2-x(PO4)3中的x | ICP、SEM/EDS | 独权或核心从权 |
| 杂质 | 磁性异物、Fe/Ni/Cr等金属杂质 | 磁性杂质测试、ICP-MS | 核心从权 |
| 水分 | 卡尔费休法水分W | Karl Fischer | 核心从权或关系式 |
| 粉体性质 | BET比表面积、振实密度 | BET、振实密度仪 | 从权或实施例变量 |
| 电学性质 | 室温离子电导率 | 阻挡电极EIS | 从权或关系式 |
| 表面状态 | Ti3+/Ti4+比例、表面缺陷指数 | XPS、EPR或等效表征 | 从权、实施例变量 |
| 分布状态 | 100um*100um截面区域内LATP颗粒面积占比偏差 | 截面SEM/EDS图像统计 | 极片层保护 |

水分测试默认写为卡尔费休法。若用户允许预测值，可将水分、NASICON相含量、离子电导率、Ti3+/Ti4+比例、比表面积、边界实施例和对比例性能作为`[P]`数据进入阶段稿。

## 3. 正极和电芯层保护层

第一保护层为LATP材料本体参数：粒径分布、相组成、元素计量比、低水分、低磁性杂质、离子电导率和表面缺陷指标。

第二保护层为正极和极片参数：正极中LATP添加量、NCM811或其他高镍正极配方比例、导电剂/粘结剂比例、面密度、压实密度、极片厚度和截面分布均匀性。

第三保护层为电芯性能：ACR、K值、首效、低温容量保持率、热箱通过结果或热失控触发温度。实施例1的关键安全结果应写为确定且较优的结果，不写成“某温度热失控或某温度通过”之类二选一表述。

## 4. 协调关系式

可优先采用以下材料因子，实测值或预测值应位于范围中心或优选中心：

```text
M=(D90/D50)*W/C
```

其中，W为水分含量或表面缺陷指数，C为正极中LATP添加量。M用于体现粒径分布、水分或缺陷水平与添加量之间的综合约束。

```text
Q=R*A/C
```

其中，R为D90/D50，A为NASICON相含量或室温离子电导率归一值，C为正极中LATP添加量。Q用于体现颗粒级配、有效导锂相和添加比例之间的耦合。

不得把浆料分散因子、固含量、pH或黏度写成权利要求1的必要协调关系式。若用户希望保留加工描述，将其放在说明书制备过程或加工稳定性段落。

## 5. 实施例和对比例写法

实施例1作为固定优选种类和优选工艺，写清LATP制备、正极配方、极片制备、电芯装配、化成和关键性能。其他实施例和对比例采用如下句式：

```text
本实施例与实施例1的区别在于：通过调整LATP烧结和后处理条件为[具体条件]，使NASICON相含量由实施例1的[原值]变为[新值][P]，并使x值变为[新值][P]，其余材料体系、制备步骤、装配条件和测试条件均与实施例1相同。
```

```text
本对比例与实施例1的区别在于：通过调整[具体制备动作或材料替换方式]，使[偏离参数]由实施例1的[原值]变为[超限值或替换值]，其余材料体系、制备步骤、装配条件和测试条件均与实施例1相同。
```

对开放式表面层布局，实施例1固定一种表面层种类，其他实施例可改变表面层组成、厚度、覆盖率或Ti3+/Ti4+比例；对比例可采用无表面层、过厚表面层或旧案高风险表面层作为偏离路线。

最终申请稿中的实施例表和对比例表只保留专利文本需要的列，例如“编号”“相对于实施例1的差异”“M值”或关键参数结果。不得保留内部管理列“数据标识”“规避类型”“证明目标”。若仍需区分预测数据，在Markdown阶段可把`[P]`附着于具体数值；正式Word阶段按用户要求清除。

## 6. 性能测试和解释

性能数据表只保留样品编号、ACR、K值、首效、低温容量保持率、热箱结果等技术列，不保留“数据标识”列。

热箱结果应为确定判定，例如“160degC通过”或“130degC热失控”，不得在同一单元格中保留替代性结果。若用户要求使用`degC`，必须在关键术语中定义：`degC表示摄氏度，与℃含义相同，用于统一英文字符表达；例如25degC表示25℃。`

性能解释采用逐变量段落，每段通常不少于100个中文字符，并包含：

1. 对比对象：实施例与对比例编号。
2. 自变量：粒径、W、C、NASICON相含量、x值、表面层、BET、Ti3+/Ti4+比例、截面面积占比偏差或极片参数。
3. 性能变化：ACR、K值、首效、低温容量保持率或热箱结果的变化。
4. 机理：离子传输通道、界面副反应、水分诱发HF、颗粒分散均匀性、热稳定性或电子/离子协同。
5. 范围结论：说明权利要求范围为什么能兼顾阻抗、低温、安全和加工可行性。

性能解释不使用箭头类符号，不只写“性能更好”。

## 7. 产品反向鉴定口径

LATP正极添加剂案件的反向鉴定可从单只未知成品电池出发。若用户取消完整前置性能复测，可直接进入安全放电和拆解流程；默认将电芯以0.2C放电至2.5V后，在惰性气氛中拆解取样。

推荐检测路径：

1. 正极片取样后用DMC或等效溶剂清洗去除残余电解液，真空干燥至恒重。
2. 用截面SEM/EDS识别LATP颗粒及其分布，统计100um*100um截面区域内LATP颗粒面积占比和面积占比偏差。
3. 用ICP测Li、Al、Ti、P元素摩尔比，归一化得到x或元素计量关系。
4. 用XRD/Rietveld测NASICON相含量和杂相含量。
5. 用Karl Fischer测水分W；用磁性杂质测试或ICP-MS测金属杂质。
6. 用BET、振实密度、EIS、XPS或EPR测比表面积、振实密度、室温离子电导率和Ti3+/Ti4+比例。
7. 将检测结果代入M或Q等关系式，形成权利要求1核心特征的判定表。

## 8. 避让旧案主线

撰写时应避免把以下路线写成核心发明点，除非用户明确要求用于对比例或排除性限定：

1. LATP基片或隔膜/固态电解质基片路线。
2. SiF4涂层作为核心差异路线。
3. LAGP/Ge壳层路线。
4. Ga-LLZO/LATP/CNT包覆路线。
5. LATP-CeO2正极复合路线。

如需出现上述内容，优先作为对比例、背景技术或排除性限定，不作为我方权利要求1的必要技术特征。
