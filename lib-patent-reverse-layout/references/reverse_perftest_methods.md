# 详细性能测试方法库 (Detailed Performance Testing Methods)

本文档定义反向专利布局技能 Step 3 阶段使用的性能测试方法库。目标：为每项性能权要提供**说明书中"性能测试方法"章节可直接引用**的标准描述。

---

## 一、撰写原则

1. **设备具体化**：写明品牌/型号（如「岛津 AGS-X 万能试验机」），或写"市售...型号..."。
2. **条件具体化**：温度、时间、电流、电压、采样点必须明确。
3. **判定具体化**：通过/不通过的边界值清楚。
4. **行业标准化**：尽可能引用 GB/T、IEC、UL、SAE 等标准号，提升权要稳定性。
5. **可复现性**：本领域技术人员看完描述能立即上手做。

---

## 二、电化学性能测试

### 2.1 比容量与首效

**测试设备**：电池测试系统（如 LANHE CT2001A、Arbin BT2000）

**电池组装**：CR2032 半电池，对锂金属负极（Φ16 mm×0.6 mm 锂片），电解液 1 M LiPF6/EC+DMC（1:1 v/v），隔膜 Celgard 2400。

**测试程序**：
```
1. 静置 12 h
2. 化成：0.05C 充至 4.3V (CV 至电流降至 0.01C) → 0.05C 放电至 3.0V
3. 测试：0.1C 充至 4.3V (CV 至 0.05C) → 0.1C 放电至 3.0V，循环 3 次
4. 取第 3 次循环数据：充电比容量、放电比容量、首次库伦效率
```

**判定**：
- 比容量 ≥ XXX mAh/g
- 首效 ≥ XX%

---

### 2.2 倍率性能

**测试程序**（同一电池连续测试）：
```
0.1C  充/放 3 次
0.2C  充/放 3 次
0.5C  充/放 3 次
1C    充/放 3 次
2C    充/放 3 次
3C    充/放 3 次
5C    充/放 3 次
0.1C  充/放 3 次（恢复测试）
```

**记录**：每个倍率第 3 次循环的放电比容量。

**判定**：通常以 5C/0.1C 比值评估倍率性能（如 5C 容量保持率 ≥ 80%）。

---

### 2.3 循环性能

**测试设备**：电池测试系统

**电池组装**：全电池（如 18650、软包），与正/负极配对。

**测试程序**：
```
化成：0.05C 充放电 1 次
正式循环：1C 充至上限电压 (CV 至 0.05C) → 1C 放电至下限电压
循环至 500 / 1000 / 2000 周
```

**记录**：每 50 周记录一次容量。

**判定**：
- 500 周容量保持率 ≥ XX%
- 1000 周容量保持率 ≥ XX%

---

### 2.4 阻抗 (EIS)

**测试设备**：电化学工作站（如 Solartron 1287/1260、PARSTAT MC、CHI 660E）

**测试条件**：
```
频率范围：100 kHz ~ 0.01 Hz
振幅：5 mV
电位：开路电位 (OCP) 静置 30 min 后
温度：25 ± 1 ℃
```

**记录**：Nyquist 图，等效电路拟合 R_s（欧姆电阻）、R_ct（电荷转移电阻）。

**判定**：R_ct ≤ XX Ω·cm²

---

### 2.5 电化学窗口

**测试设备**：电化学工作站，三电极体系（工作电极 SS 不锈钢，对/参比电极 Li 金属）

**测试条件**：
```
扫描速率：0.5 mV/s
扫描范围：0 ~ 6 V vs Li/Li+
温度：25 ℃
```

**判定**：以电流密度 < 0.1 mA/cm² 为稳定窗口边界。

---

### 2.6 高倍率平台电压差与 40A 温度辅助测试

**适用对象**：圆柱、软包或方形锂离子电池的高倍率低极化效果评价。适用于权要或说明书中限定“平台电压差”“高倍率平台电压差增量斜率”“40A 温度辅助观察”等指标的案件。

**统一定义**：平台电压差为同一 SOC 或同一容量区间下，低倍率参考平台电压与对应倍率平台电压之差。正式申请文本中只能保留这一种口径，不得在同一组数据中混用其他定义。

**测试设备**：动力电池测试系统或同等电流精度的电池充放电测试系统；温度记录可采用贴片热电偶或红外热像仪。

（步骤2.6.1）样品准备：取实施例、对比例及对照样电池作为待测样品。每组样品优选不少于 3 只；如因反向拆解或样品数量受限仅能取得单只样品，应在数据表中注明样品数量。测试前记录样品编号、尺寸、质量、开路电压、交流内阻和额定容量。

（步骤2.6.2）标准测试条件：测试环境温度为 25±2℃，优选为 25±1℃；测试环境中无主动气流，电池水平放置，距相邻物体距离不小于 50 mm。各组样品采用相同测试柜、相同电流量程、相同夹具、相同导线连接方式和相同静置条件。测试前，将待测电池调节至相同初始 SOC 窗口，并静置不少于 2 h。

（步骤2.6.3）低倍率参考平台建立：对待测样品进行标准充电后，以低倍率电流进行参考放电，获得低倍率参考放电曲线。所述低倍率电流优选为 0.1C，也可为 0.05C 至 0.2C 范围内的任一电流；同一批测试中低倍率电流应保持一致。根据低倍率参考放电曲线，获得同一 SOC 或同一容量区间下的低倍率参考平台电压，记为 Vref。

（步骤2.6.4）倍率放电曲线测试：将待测样品恢复至与步骤2.6.3相同的初始 SOC 窗口，分别以 10A、20A、30A、35A 和 40A 进行恒流放电测试，并记录放电电压-时间曲线、电流-时间曲线、容量-时间曲线和温度-时间曲线。相邻倍率测试之间应进行标准充电、静置和温度恢复；若前一倍率测试后样品表面温度与环境温度之差大于 1℃，应继续静置至不大于 1℃后再进行下一次测试。电压和电流采样频率优选不低于 1 Hz。

（步骤2.6.5）平台区间选取：在低倍率参考放电曲线和对应倍率放电曲线中选取相同 SOC 区间或相同放电容量区间作为平台区间。平台区间应避开放电开始初期的瞬态压降区和放电末期的电压快速下降区；未特别说明时，可选取 20%SOC 至 80%SOC 对应区间，或选取与该 SOC 范围相对应的容量区间。同一组实施例、对比例和对照样应采用同一平台区间。

（步骤2.6.6）平台电压差计算：对任一放电电流 I，在平台区间内计算对应倍率平台电压 VI，并按下式计算平台电压差 Delta V_I：

```text
Delta V_I = Vref - VI
```

其中，I 可为 10A、20A、30A、35A 和 40A；Vref 为同一 SOC 或同一容量区间下的低倍率参考平台电压，VI 为同一 SOC 或同一容量区间下的对应倍率平台电压。若采用平均平台电压，则 Vref 和 VI 均为平台区间内的平均电压；若采用指定 SOC 点电压，则 Vref 和 VI 均为同一 SOC 点下的电压。

（步骤2.6.7）平台电压差增量斜率计算：为评价高倍率区间内平台电压差随放电电流增加的变化程度，计算 20A 至 40A 平台电压差增量斜率 S20-40：

```text
S20-40 = (Delta V_40A - Delta V_20A) / 20
```

其中，Delta V_40A 为 40A 平台电压差，Delta V_20A 为 20A 平台电压差，S20-40 的单位为 V/A。

（步骤2.6.8）性能判定：高倍率低极化案件可根据已确认实测数据设置 40A 平台电压差阈值和 S20-40 范围。阈值必须以实测产品为中心或优选中心，并由实施例、边界实施例、对比例或机理解释支撑。不得只用温度指标替代平台电压差作为低极化核心效果。

（步骤2.6.9）40A 温度辅助测试：在 40A 放电测试过程中，采用贴片热电偶或红外热像仪记录电池最大面积侧的表面温度。采用贴片热电偶时，测温点位于壳体轴向高度 H 的 1/2±2 mm 位置，并在周向上避开壳体焊缝、极耳焊接热影响区或壳体结构异常区域至少 30°。采用红外热像仪时，应记录最大面积侧中心区域、靠近极耳区域和边缘区域的最高温度。

（步骤2.6.10）40A 最高温度记录：自 40A 放电开始至放电结束后 300 s 内，连续记录电池表面温度曲线；取该时间窗口内的最高表面温度作为 40A 温度。该温度仅作为辅助效果说明，不作为独立权利要求的默认核心判定条件；除非用户明确选择温升控制为本案核心，不采用放电后冷却拟合参数、热环境校正参数或电热耦合温升指数作为默认评价模型。

（步骤2.6.11）结构参数复核：完成性能测试后，可对同批样品或对应拆解样品进行极片展开、长度测量、截面测厚、面密度测量、隔膜截面表征和材料表征，以确认样品结构变量与对应实施例或对比例一致。复核项目应覆盖独立权利要求中的核心结构参数。

（步骤2.6.12）数据有效性确认：若同组样品中 40A 平台电压差的最大值与最小值之差大于 0.010 V，或放电过程中出现夹具接触异常、电压采样中断、异常温升、保护动作或明显容量偏离，应复核测试连接和样品状态，并重新测试或增加样品数量。基于已有实测数据和结构机理推理得到的数据，必须在数据属性栏标注 `[P]`。

## 三、热性能测试

### 3.1 热收缩率

**适用对象**：胶带、隔膜、涂层。

**测试设备**：恒温烘箱（精度 ±1 ℃），高精度直尺或显微测量仪。

**样品制备**：
```
1. 切取 100 mm × 100 mm 方形样品
2. 标记 MD 方向（机器方向）与 TD 方向（横向）
3. 测量初始尺寸 L0_MD、L0_TD
```

**测试程序**：
```
温度：85 ℃ / 105 ℃ / 120 ℃ / 150 ℃（按权要选定）
时间：30 min / 60 min（按权要选定）
放置：玻璃板上自由放置（无张力）
冷却：取出后室温冷却 30 min
测量：L1_MD、L1_TD
```

**判定**：
- 热收缩率_MD = (L0_MD - L1_MD) / L0_MD × 100% ≤ XX%
- 热收缩率_TD = (L0_TD - L1_TD) / L0_TD × 100% ≤ XX%

**示例**（本次案例权 11）：「在 85℃、30 min 条件下的热收缩率不大于 2.0%」。

---

### 3.2 热分解温度 (TGA)

**测试设备**：热重分析仪（如 TA Q500、NETZSCH STA 449）

**测试条件**：
```
样品量：5-10 mg
气氛：N2 或 Ar（流量 50 mL/min）
升温速率：10 ℃/min
温度范围：30 ~ 600 ℃
```

**判定**：以失重 5% 对应温度作为热分解起始温度 (T5%)。

---

### 3.3 DSC 热行为

**测试设备**：差示扫描量热仪（如 TA Q200、NETZSCH DSC 214）

**测试条件**：
```
样品量：3-5 mg
气氛：N2，流量 50 mL/min
升温速率：10 ℃/min
温度范围：-40 ~ 300 ℃
```

**记录**：玻璃化转变温度 Tg、熔点 Tm、结晶峰 Tc、放热峰位置与峰强。

---

## 四、力学性能测试

### 4.1 拉伸模量与拉伸强度

**测试设备**：万能材料试验机（如岛津 AGS-X、Instron 5965）

**样品制备**：哑铃型 GB/T 1040 II 型或 ISO 527-2 1BA 型

**测试条件**：
```
拉伸速率：50 mm/min（薄膜类）；5 mm/min（厚膜类）
夹具间距：50 mm
温度：23 ± 2 ℃，相对湿度 50 ± 5%
```

**计算**：
- 拉伸模量 E = (σ2 - σ1) / (ε2 - ε1)，取 ε = 0.05% ~ 0.25% 区间
- 拉伸强度 σ_max
- 断裂伸长率 ε_b

**示例**（本次案例权 5）：「拉伸模量为 1.5GPa~6.0GPa」。

---

### 4.2 剥离强度

**测试设备**：剥离试验机（如三思 UTM4204）

**测试条件**：
```
剥离角度：180° 或 90°
剥离速率：100 mm/min（胶带常用）；300 mm/min（极片涂层）
样品宽度：25 mm（胶带）；15 mm（极片）
基底：SUS304 钢板（胶带）；铝箔/铜箔（极片）
```

**记录**：单位宽度上的平均剥离力 (N/m 或 N/25mm)。

---

### 4.3 压实密度

**适用对象**：极片、活性材料。

**测试设备**：万能材料试验机 + 压片模具，或专用压实密度仪（如 Sintech 二代）

**测试程序**：
```
1. 称取一定量 (m) 的活性材料粉体
2. 加入 Φ13 mm 模具
3. 压制：压力 10 MPa / 20 MPa / 30 MPa / 50 MPa（多档）
4. 保压 30 s
5. 卸压，测量片厚 (h)
```

**计算**：压实密度 ρ = m / (π × r² × h)

---

## 五、化学/电化学稳定性测试

### 5.1 耐电解液性

**测试程序**：
```
1. 将样品（胶带/隔膜/涂层）裁切为 50×50 mm
2. 浸泡于 1 M LiPF6/EC+DMC (1:1 v/v) 中
3. 温度：25 ℃ / 60 ℃（按权要选定）
4. 时间：72 h
5. 取出 → DMC 漂洗 → 真空干燥
6. 测量重量变化、外观（开裂/起皮）、拉伸性能变化
```

**判定**：
- 重量变化 ≤ ±3%
- 拉伸强度保持率 ≥ 80%
- 无开裂、起皮、变色

---

### 5.2 高温存储

**测试程序**：
```
1. 满充电池（4.3V CCCV 至 0.05C）
2. 60 ℃ 烘箱存储 7 / 14 / 28 d
3. 室温恢复 6 h
4. 测量：开路电压、容量恢复率（与存储前对比）
```

**判定**：容量保持率 ≥ XX%，容量恢复率 ≥ XX%。

---

## 六、安全性能测试

### 6.1 针刺测试

**测试设备**：针刺测试机（如 IEC 62133-2 标准设备）

**测试条件**：
```
针径：Φ3 / Φ5 / Φ8 mm（按标准）
材质：高速钢 (HSS) 或不锈钢
针尖角度：45° ~ 60°
针刺速度：10 ~ 80 mm/s
针刺位置：电池几何中心
电池状态：满充（4.3V）
环境温度：25 ℃
```

**判定**：
- 通过：不起火、不爆炸
- 不通过：起火、爆炸、外壳破裂喷射

**记录**：通过率 (n/N, %)。

---

### 6.2 过充测试

**测试条件**：
```
充电电流：1C 或 3C
截止电压：上限电压的 1.5 倍 或 4.6 V
持续时间：直到电池失效 (起火/爆炸/短路)，或 12 h 安全通过
```

**判定**：通过 = 不起火不爆炸。

---

### 6.3 热冲击

**测试条件**：
```
温度：130 ℃ / 150 ℃ / 180 ℃（按权要）
保持时间：30 min
升温速率：5 ℃/min
环境：恒温烘箱
```

**判定**：通过 = 不起火不爆炸。

---

### 6.4 挤压

**测试设备**：挤压试验机

**测试条件**：
```
挤压头：Φ32 mm 圆柱
挤压速度：5 mm/s
挤压方向：电池长轴 × 短轴交叉方向
终止条件：力达 13 kN 或电池变形量 50%
```

**判定**：通过 = 不起火不爆炸。

---

## 七、视觉/光谱鉴别测试

### 7.1 颜色

**测试设备**：分光测色仪（如 Konica Minolta CM-700d）

**测试条件**：标准光源 D65，10° 视角，反射模式。

**记录**：CIELAB 色坐标 L*、a*、b*；与标准色卡比对。

**示例**（本次案例权 10）：「黄色聚酰亚胺胶带」「蓝色聚酯胶带」。

---

### 7.2 FTIR 鉴别

**测试设备**：傅里叶变换红外光谱仪（如 Bruker Vertex 70、Thermo Nicolet iS50）

**测试条件**：
```
分辨率：4 cm⁻¹
扫描次数：32 / 64
波数范围：4000 ~ 400 cm⁻¹
模式：ATR 或 KBr 压片
```

**判定**：通过特征峰位（如 PI 在 1720 cm⁻¹ 的酰亚胺 C=O 伸缩、1380 cm⁻¹ 的 C-N 伸缩）识别基材种类。

---

### 7.3 XRD 鉴别

**测试设备**：X 射线衍射仪（如 Rigaku SmartLab、Bruker D8 Advance）

**测试条件**：
```
辐射源：Cu-Kα (λ = 1.5418 Å)
扫描范围：5° ~ 80° (2θ)
扫描步长：0.02°
扫描速率：5°/min 或 10°/min
```

**判定**：通过衍射峰位与晶面指数识别晶相（如 LiCoO2 的 (003)、(101) 峰）。

---

## 八、性能数据汇总表撰写模板

```
表 1 实施例及对比例性能数据汇总（[P] 为基于电化学原理的预言性数据）

| 编号 | [参数1] | [参数2] | ... | 能量密度(Wh/kg) | 针刺(通过/起火 %) | 500周保持率(%) | [其他性能] |
|------|---------|---------|-----|-----------------|-------------------|-----------------|--------------|
| 实施例1   | ...     | ...     |     | 178             | 100/0             | 99.2            |              |
| 实施例2 [P]| ...    | ...     |     | 178             | 99/0              | 99.1            |              |
| ...        |         |         |     |                 |                   |                 |              |
| 对比例1 [P]| ...    | ...     |     | 172             | 90/10             | 95.5            |              |
| ...        |         |         |     |                 |                   |                 |              |
```

**关键要素**：
1. 实测数据不标注 `[P]`
2. 预言性数据标注 `[P]`
3. 实施例最差值 < 对比例最好值（形成清晰分界）
4. 每个权要的关键性能至少在表中体现 1 列
